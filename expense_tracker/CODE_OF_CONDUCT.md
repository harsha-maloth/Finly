# Finly Code of Conduct

Be friendly and respectful. Finly is a small, community-driven project focused on learning and privacy-first personal finance. Everyone is welcome.

Expected behavior
- Treat contributors with kindness and professionalism.
- Use inclusive language.
- Provide constructive feedback.

Unacceptable behavior
- Harassment, discrimination, or personal attacks.
- Posting private information or attempting to de-anonymize contributors.

Enforcement
If you experience or witness unacceptable behavior, please open an issue or contact the project maintainer on GitHub.

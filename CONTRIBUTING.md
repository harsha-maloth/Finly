# Contributing to Finly

Thanks for your interest in contributing to Finly. This is a beginner-friendly, privacy-first project. Please respect the offline-only goals when proposing changes.

Steps to contribute
1. Fork the repository and create a branch for your change.
2. Keep changes small and focused. Run a quick syntax check:

```bash
python -m py_compile expense_tracker/*.py
```

3. Open a pull request with a clear description.

Reporting bugs
- Provide clear reproduction steps, Python version, OS, and any traceback.

Note
- Do not add network calls or telemetry. Finly is intentionally local-only.
